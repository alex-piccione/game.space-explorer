# Space Explorer

A lightweight, first-person cockpit-view space exploration game. Experience the sensation of piloting a starship through a procedurally generated universe, designed for quick breaks and immersive "zen" moments.

## Concept
The game is played from a "cockpit view," where the player sees the interior of a spaceship cabin, with stars, planets, and nebulae passing by outside. The goal is to provide a relaxing yet engaging sense of movement through space.

## Tech Stack
- **Engine:** [Three.js](https://threejs.org/) (WebGL)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Language:** TypeScript
- **Platform:** Web Browser
